# COMSOL Bridge

This folder will hold MATLAB/COMSOL LiveLink scripts for converting binary masks
into full 3D COMSOL models.

Planned files:

- `mask_to_comsol_geometry.m`
- `run_frequency_sweep.m`
- `extract_sparams.m`
- `render_field_plots.m`

Initial validation order:

1. Baseline 3/3/3 CPS.
2. Existing strong-stub candidate from `play/real3`.
3. Route B generated candidate.
4. Ablation candidates.
